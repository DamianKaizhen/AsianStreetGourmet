Forgot password?

1. Open the Vercel dashboard for this project → Settings → Environment Variables.

2. Add a new variable: name ADMIN_RECOVERY_MODE, value true. Save and redeploy.

3. Come back here and sign in with the ORIGINAL password (the one Damian set at first deploy — should be in a password manager).

4. Once you're in, set a new password from the "Change password" section, then DELETE the ADMIN_RECOVERY_MODE variable from Vercel.

The current password cannot be retrieved — passwords are stored as one-way hashes for security. Recovery resets to the env-var password instead.
